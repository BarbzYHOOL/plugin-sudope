## sudope Plugin for Fish Shell

### About

A Fish port of om-my-zsh's [sudo plugin]

### Install

With [Oh-My-Fish]:
```fish
omf install sudope
```

### Usage

Press `Ctrl`+`s` once to add 'sudo' (normal behaviour),
press it twice to remove it from the beginning of the line.
